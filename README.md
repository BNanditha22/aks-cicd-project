# AKS CI/CD DevOps Project 🚀

## Overview
This project demonstrates a Flask application containerized using Docker and prepared for Kubernetes deployment.

## Tech Stack
- Python (Flask)
- Docker
- Kubernetes (AKS - upcoming)
- CI/CD (GitHub Actions - upcoming)

## Run Locally
pip install -r requirements.txt  
python app.py  

## Run with Docker
docker build -t myapp .  
docker run -p 5001:5000 myapp
