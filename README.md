## 📌 Project Overview

This project demonstrates an end-to-end DevOps pipeline using Azure and Kubernetes.

A containerized Flask application is built using Docker, pushed to Azure Container Registry (ACR), and deployed on Azure Kubernetes Service (AKS) with a fully automated CI/CD pipeline using GitHub Actions.

---

## 🛠️ Tech Stack

- ☁️ Azure (AKS, ACR)
- 🐳 Docker
- ☸️ Kubernetes
- 🔄 GitHub Actions (CI/CD)
- 🐍 Python (Flask)
